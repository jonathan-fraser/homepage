The distance set problem

The distance set problem is a famous and well-studied problem in geometric measure theory. It stems from a paper of Kenneth Falconer from 1985 and asks about the relationship between a subset of Euclidean space and its distance set. The distance set associated with a given set is simply the set of all distances realised between pairs of points in the set. For example, if the Hausdorff dimension of a Borel set in d-dimensional Euclidean space is at least d/2, then it is conjectured that the Hausdorff dimension of the distance set should be 1 (which is maximal, since the distance set is a set of real numbers). I solved the Assouad dimension analogue of the distance set problem for subsets of the plane here:

            Journal of the London Mathematical Society, 107, (2023), 777-797..
I have used Fourier analytic tools to study the distance set problem:

I have used ergodic theoretic tools, such as CP-chains, to study the distance set problem. For example, this approach was used to obtain partial results on the Assouad dimension problem here:
    ► Distance sets, orthogonal projections, and passing to weak tangents, arXiv
            Israel Journal of Mathematics, 226, (2018), 851-875.
The ergodic theoretic approach is very useful when the underlying fractal is dynamically defined. I used this approach to solve the Hausdorff dimension version of the distance problem for certain self-affine and self-conformal sets here:
    ► Scaling scenery of (×m,×n) invariant measures (with A. Ferguson & T. Sahlsten), arXiv
            Advances in Mathematics, 268, (2015), 564-602.
    ► Micromeasure distributions and applications for conformally generated fractals (with M. Pollicott), arXiv
            Mathematical Proceedings of the Cambridge Philosophical Society, 159, (2015), 547-566.
